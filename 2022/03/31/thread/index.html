线程方法

void join() 等待该线程终止
static void sleep (long millis) 在指定的毫秒数内让当前正在执行的线程体休眠，不会释放锁
static void yield () 暂停当前正在执行的线程对象，并执行其他线程，让cpu重新调度
setPritority() 更改线程的优先级
boolean isAlive() 测试线程是否处于活动状态
